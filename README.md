# WiFi-Scanner

Backend for scanning the home WiFi network. Required Nmap. 

## Install

### Installing Nmap

Linux:
```
sudo apt-get install nmap
```

Windows:   
Download from Nmap's official website: https://nmap.org/download.html 

A restart may be required on Windows.
